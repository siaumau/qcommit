📦 VS Code Marketplace 发布                                                                                                                         
                                                                                                                                                        当前情况                                                                                                                                            
                                                                                                                                                      
  - VS Code Marketplace 自动使用 README.md
  - README-EN.md 不会自动显示在 Marketplace

  方案 1️⃣- 推荐（GitHub优先）

  保持现在的方式，发布时只需更新 README.md：

  📁 项目根目录
  ├── README.md (中文 + 语言切换链接)
  ├── README-EN.md (英文完整版)
  └── package.json

  Marketplace 显示：README.md（中文）
  - 用户可以点击链接 → 看 GitHub 上的 README-EN.md

  ---
  方案 2️⃣- 双重主文档

  可以在 package.json 中添加字段指向英文版本：

  {
    "readme": "README.md",
    "homepage": "https://github.com/你的用户名/qcommit#readme"
  }

  ---
  🚀 发布流程

  每次发布时：

  # 1. 同时更新两个文件
  # README.md (中文)
  # README-EN.md (英文)

  # 2. 更新版本号
  # 修复版本号 (0.0.1 → 0.0.2) - 小bug修复
  npm version patch          
                                                                                                                                                      
  # 次版本号 (0.0.1 → 0.1.0) - 新功能
  npm version minor

  # 主版本号 (0.0.1 → 1.0.0) - 重大更新
  npm version major

  # 3. 编译
  npm run compile

  # 4. 发布
  vsce publish

  ---
  📝 我的建议

  ✅ 最简单：
  - GitHub 上保持两个 README
  - Marketplace 上只显示中文（README.md）
  - 中文 README 的顶部有英文链接

  这样用户可以：
  - 在 Marketplace 看中文版
  - 在 GitHub 切换英文版本
  - 两版本都完整
