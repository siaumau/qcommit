# VS Code 自製 AI Commit Message Extension 開發指南

## 目標

建立一個 **VS Code Extension**，可以：

1.  讀取 `git diff --staged`
2.  將 diff 傳送到你的 AI API
3.  生成 commit message
4.  自動填入 VS Code 的 commit 輸入框

這樣就不需要使用外掛預設的 OpenAI API key，而是可以使用：

-   自己的 OpenAI API key
-   自架 API
-   Ollama
-   vLLM
-   OpenRouter
-   企業內部模型服務

# Step 2 取得 git staged diff

``` ts
import { execSync } from "child_process";

export function getStagedDiff(): string {
  try {
    const diff = execSync("git diff --cached").toString();
    return diff;
  } catch (error) {
    return "";
  }
}
```

------------------------------------------------------------------------

# Step 3 呼叫 AI API

``` ts
import fetch from "node-fetch";

export async function generateCommit(diff: string) {

  const prompt = `
請根據以下 git diff 產生一則簡潔的 commit message。
使用 Conventional Commits 格式。

${diff}
`;

  const response = await fetch("https://api.openai.com/v1/chat/completions", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "Authorization": `Bearer YOUR_API_KEY`
    },
    body: JSON.stringify({
      model: "gpt-4o-mini",
      messages: [{role:"user",content:prompt}]
    })
  });

  const data = await response.json();

  return data.choices[0].message.content;
}
```

------------------------------------------------------------------------

# Step 4 寫入 VS Code commit box

``` ts
import * as vscode from "vscode";

export async function setCommitMessage(message:string){

  const gitExtension =
    vscode.extensions.getExtension("vscode.git")?.exports;

  const git = gitExtension.getAPI(1);
  const repo = git.repositories[0];

  repo.inputBox.value = message;

}
```

------------------------------------------------------------------------

# Step 5 建立 command

``` ts
vscode.commands.registerCommand(
  "commitAI.generateMessage",
  async () => {

    const diff = getStagedDiff();

    if(!diff){
      vscode.window.showErrorMessage("沒有 staged changes");
      return;
    }

    const message = await generateCommit(diff);

    setCommitMessage(message);

  }
);
```
